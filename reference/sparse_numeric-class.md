# Sparse numeric vector

S4 class representing a sparse numeric vector.

S4 class representing a sparse numeric vector.

## Slots

- `value`:

  Numeric vector of non-zero entries.

- `pos`:

  Integer vector of 1-based positions of non-zero entries.

- `length`:

  Integer scalar giving the full length of the dense vector.
